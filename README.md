# Ecommerce Catalog - Final Task VIX Core Initiative

Project ini dibuat sebagai Final Task Frontend Engineer menggunakan **Vue.js versi 2**, di mana peserta diminta:

1. Mendesain halaman katalog produk.
2. Menghubungkan UI dengan API FakeStoreAPI.
3. Melakukan filter kategori:
   - men's clothing
   - women's clothing
4. Jika kategori selain 2 di atas → tampil halaman "Unavailable Product".
5. Menggunakan Vanilla CSS (tanpa framework).
6. Melakukan class binding berdasarkan kategori.

---

## Teknologi
- Vue 2
- Javascript ES6
- Fetch API
- Vanilla CSS
- Git / GitHub

---

## API
Menggunakan FakeStoreAPI:

https://fakestoreapi.com/products/{index}

Index valid: **1–20**.
